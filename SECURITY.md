# Security Policy

## Supported Versions

We support the latest released version of PreReview. Please update to the most recent version before reporting issues.

## Reporting a Vulnerability

- Email: security reports to emilushi@gmail.com
- Please include:
  - Description of the issue and impact
  - Steps to reproduce or proof-of-concept
  - Affected version (commit or tag) and environment details
- You will receive an acknowledgement within 72 hours.

## Disclosure

We aim to provide a fix and release within a reasonable timeframe. Please do not publicly disclose vulnerabilities until a fix is available and users have had time to update.
